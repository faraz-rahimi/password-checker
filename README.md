# Password Checker

A simple and responsive Password Checker built with HTML, CSS, and JavaScript.

## ✨ Features

- Validate password input
- Floating label input
- Press Enter to submit
- Responsive layout
- Success and error messages

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (DOM)
